About me

I am a Postdoctoral researcher in the Mathematics department of University at Buffalo. I work with Dr. Naoki Masuda and Dr. A. Erdem Sariyuce on fraud detection in financial networks using network motifs. This project is funded by J.P.Morgan AI research award.

I completed my PhD from the Computer Science department of University of Rochester under the supervision of Dr. Daniel Štefankovič. In my PhD thesis I worked on Approximating Partition Functions of Spin Systems and Its Applications.

I also work on developing probabilistic models for various applications in Machine Learning (e.g., Fairness in AI, Mental Health Prediction from Web Data, Neural Network Pruning etc.).

Mental Health Detection from Web Data

Online behavior leaves a digital footprint that can be analyzed to reveal our cognitive and psychological state through time. Recognizing these subtle cues can help identify different aspects of mental health, such as low self-esteem, depression, suicide ideation and anxiety. In this project we identify such traits of mental health using probabilistic models on Google search data.







Fairness in Public Speaking

With the recent trend of applying machine learning in every aspect of human life, it is important to incorporate fairness into the core of the predictive algorithms. We address this problem by detecting and mitigating unfairness in speech rating using causal model and deep learning techniques. We use the TED talks as an input repository of public speeches and aim to achieve fairmess with respect to sensitive attributes of the speakers, e.g. gender and race.





Neural Network Pruning

In recent times neural networks have achieved state of the art accuracy for most ML tasks. However, these networks often contain millions of parameters. Therefore reducing the number of parameters of the neural network (also known as neural network pruning) has recently gained much importance amongst researchers.In this project we design neural network pruning method using determinantal point process (DPP).





Estimating Partition Function in Dense Graphs

Spin systems have been extensively studied in physics, mathematics and machine learning. A spin system consists of a collection of sites which are the vertices of an underlying connected graph. A configuration of the spin system is an assignment of one of a finite set of spins to each site. The sites interact locally, according to the interaction parameter, such that different combinations of spins on neighboring sites have different relative likelihoods. This interaction gives rise to a well-defined probability distribution over configurations of any finite subset (region) of the sites. The normalization factor of these distributions (also known as partition fucntion) is a key factor for sampling/counting these configurations. In this project we estimate the partition functions for dense graphs.
